# NexCode-AI 🚀

An all-in-one developer workspace combining a real-time collaborative code editor, video conferencing, and AI tools to eliminate context switching and boost productivity.

## ✨ Features
* **Real-Time Code Editor:** Write and execute code collaboratively with peers.
* **Video Conferencing:** Communicate face-to-face natively within the workspace.
* **AI Assistant:** Built-in AI integration for problem-solving and debugging.

## 🛠️ Tech Stack
* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Real-Time Collaboration:** Socket.io
* **Code Editor & Execution:** CodeMirror, OneCompiler 
* **Video Conferencing:** ZegoCloud
* **AI Integration:** Gemini API

## 🔍 Problem Statement

In modern software development, remote collaboration often requires developers to constantly switch between multiple single-purpose applications:
* **Context Switching:** Developers have to use one app for writing code (like an IDE), a second app for communication (like Zoom, Google Meet, or Microsoft Teams), and a third app for AI assistance (like a browser tab with ChatGPT or Gemini).
* **Time & Efficiency Loss:** Moving back and forth between screens, copy-pasting code snippets into chat windows, and sharing external meeting links breaks the developer's focus, leading to a fragmented workflow and wasted time.
* **Collaboration Barriers:** Traditional video calls don't offer native, real-time code synchronization, making pair programming or collaborative debugging clunky and disjointed.

---

## 💡 The Solution: NexCode-AI

**NexCode-AI** solves these inefficiencies by creating a single, unified virtual workspace. Instead of managing three separate tools, everything a team needs to collaborate is integrated natively into one tab:

1. **Native Integration:** By bringing real-time code editing, video conferencing, and AI debugging tools into a single platform, it completely eliminates context switching, allowing developers to maintain their flow state.
2. **Seamless Pair Programming:** Teams can code together dynamically using CodeMirror and Socket.io while actively discussing logic over high-quality, built-in ZegoCloud video streams—no external links required.
3. **Instant AI Intelligence:** With the built-in Gemini API, developers can ask questions, debug errors, or generate boilerplate code right alongside their live editor, getting answers exactly where their code lives.
